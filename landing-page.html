Features

🌐

Works everywhere

Any text field on any website. Gmail, Docs, social media, messaging apps, and more.

🇬🇧

Preserves English

English words stay as-is. Mixed Greeklish and English? No problem.

⌨️

Custom shortcuts

Set your own keyboard shortcut or use the default Ctrl+Shift+G.

🔒

Private and secure

No data collection, no tracking. Your API key stays on your device.

Get started in 30 seconds

1
Install the extension from the Chrome Web Store
2
Get your API key from console.anthropic.com
3
Click the extension icon, paste your key, and hit Save
4
Type Greeklish anywhere and press Ctrl+Shift+G
Translation costs are minimal - fractions of a cent per request using Claude Haiku.
